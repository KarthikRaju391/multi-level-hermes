# Agent instructions for this repo

This repository is a publishable Hermes starter template for separated personal/work/insights contexts.

- Do not commit secrets, OAuth caches, runtime memory, exported emails, or ticket data.
- Prefer read-only integrations and Markdown summaries.
- Do not add tools that can send, delete, archive, post, close, assign, or mutate external systems without explicit user approval.
- Keep personal and work memory paths separate. The insights profile may read both and write only derived summaries.
