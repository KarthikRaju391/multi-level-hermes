# Runtime memory template

`scripts/bootstrap.sh` copies this directory to `memory/` for local runtime use.

The generated `memory/` directory is ignored by git because it will contain personal and work summaries.

## Scopes

- `personal/` — personal inbox summaries, daily notes, personal projects, personal decisions.
- `work/` — work inbox summaries, ticket summaries, work projects, work decisions.
- `insights/` — derived cross-context daily/weekly reviews and patterns.

Store summaries and source pointers by default. Avoid raw message dumps.
